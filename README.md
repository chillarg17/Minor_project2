# Minor_project2

SpendDNA is a Python-based financial analytics project that transforms raw UPI and bank transaction data into meaningful insights through data cleaning, vendor extraction, spending analysis, anomaly detection, and spending archetype identification.

Project Constraints
Used only Python, Pandas, and NumPy.
No Regex (re) for vendor extraction.
No Scikit-learn (sklearn) or any machine learning libraries.
No SciPy for statistical calculations.
No visualization libraries such as Matplotlib, Seaborn, or Plotly.
Implemented manual Z-score calculation for anomaly detection.
Vendor extraction performed using string methods and dictionary mapping.
Category tagging implemented using custom mapping logic.
Monthly analysis performed using groupby() and pivot_table().
Time-of-day analysis implemented using Pandas datetime operations.
Code written using basic Python programming concepts, Pandas, and NumPy only, following the project guidelines.
